# Assets Guide

Drop your actual brand files here to match Sound Guess exactly:

- thunder-sans.woff2  -> custom headline font (rename if needed and update index.css)
- logo.svg or logo.png -> app header + favicon (we included a placeholder logo.svg)
- bg-texture.png       -> optional noise or pattern; add as a body background layer if desired

After replacing files, restart Vite dev server to see changes.
